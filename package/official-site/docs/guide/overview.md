# 概览

TrailSnap（行影集）是一个面向“旅行照片与行程记忆”的**相册应用**，核心目标是把照片从文件堆里变成可检索、可统计、可复盘的**数据中心**，它保留了你的所有**出行记忆**。

## 一句话看懂

- 你提供照片目录，TrailSnap 负责索引、预览、检索与可视化
- 后端负责媒体/任务/统计/票据等业务，AI 微服务负责识别能力
- 前端 Web 应用用于日常使用，官网用于文档与演示

## 关键路径（推荐体验顺序）

- 第一步：按 [安装指南](/docs/guide/install) 跑起来
- 第二步：导入照片并触发索引/任务（生成预览、提取元数据）
- 第三步：体验 AI 能力（OCR/人脸/分类/车票识别）
- 第四步：用统计与年度报告复盘出行轨迹
