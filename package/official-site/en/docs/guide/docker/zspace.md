# Zspace Deployment

This tutorial is based on TrailSnap's Docker Compose deployment method, suitable for deployment and long-term operation on Zspace devices. The goal is to let TrailSnap run stably as a home service while mounting the photo library into the container for scanning and indexing.

It is recommended to read the generic chapter first: [/en/docs/guide/docker/](/en/docs/guide/docker/)
