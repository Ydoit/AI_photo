# Fnos Deployment

This tutorial is for deploying TrailSnap on Fnos via Docker Compose. Interfaces may vary slightly between versions, but the key points are consistent: application data persistence, photo directory mounting, port planning, and permission verification.

It is recommended to read the generic chapter first: [/en/docs/guide/docker/](/en/docs/guide/docker/)
